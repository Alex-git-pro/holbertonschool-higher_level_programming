The ``MyList`` class
=====================

Using ``MyList.print_sorted``
-----------------------------

First, import the class and create an instance of `MyList`:
    >>> MyList = __import__('1-my_list').MyList
    >>> my_list = MyList()

Test with a list of integers:
    >>> my_list.append(5)
    >>> my_list.append(3)
    >>> my_list.append(9)
    >>> my_list.append(1)
    >>> my_list.print_sorted()
    [1, 3, 5, 9]

Test with a list of integers in descending order:
    >>> my_list.append(2)
    >>> my_list.append(8)
    >>> my_list.append(4)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5, 8, 9]

Test with an already sorted list:
    >>> my_list.append(10)
    >>> my_list.append(7)
    >>> my_list.print_sorted()
    [1, 2, 3, 4, 5, 7, 8, 9, 10]

Test with an empty list:
    >>> empty_list = MyList()
    >>> empty_list.print_sorted()
    []

Test with duplicate integers in the list:
    >>> my_list.append(5)
    >>> my_list.append(3)
    >>> my_list.append(3)
    >>> my_list.print_sorted()
    [1, 2, 3, 3, 4, 5, 5, 7, 8, 9, 10]

Test with a list of negative and positive integers:
    >>> my_list.append(-1)
    >>> my_list.append(-5)
    >>> my_list.append(4)
    >>> my_list.print_sorted()
    [-5, -1, 1, 2, 3, 3, 4, 4, 5, 5, 7, 8, 9, 10]

Test with a mix of integers and non-integer types:
    >>> my_list.append(10.5)
    >>> my_list.append("another string")
    Traceback (most recent call last):
    TypeError: '<' not supported between instances of 'float' and 'str'
